## 简介

transpose文件夹为用户展示了SDAA C编程模型所提的矩阵转置操作接口的使用方式。

## 支持的Teco芯片

T1

## 支持的操作系统

Linux

## 支持的CPU架构

x86_64，Loongnix，aarch64，SW_64

## 环境依赖

SDAAC_examples的运行依赖以下组件，可以参考[环境安装手册](http://docs.tecorigin.com/release/software_installation/)进行安装。

- TecoDriver
- TecoToolKit

## 涉及的API接口

该文件夹内的示例主要涉及SDAA C编程模型所提供的转置操作接口，您可以通过SDAA C编程模型中的转置操作章节学习转置接口的使用方法：
- [SDAA C编程指南-转置操作](http://docs.tecorigin.com/release/sdaac/v1.16.0/#c25f313081c911ee93790242ac110008)：介绍转置操作的各个接口。

## 编译运行

编译运行方式是在SDAAC_examples目录下执行run_customops.sh，具体步骤如下：

```
bash run_customops.sh transpose
```
